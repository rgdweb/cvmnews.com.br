## 📋 Como Funciona

1. Baixe o instalador do seu plano
2. Instale no computador
3. Abra o programa e veja o **Código de Solicitação**
4. Envie o código + seu **CPF** para o vendedor
5. Receba a **Chave de Ativação**
6. Ative e comece a usar!
